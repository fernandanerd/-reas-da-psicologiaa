# HTML : Quais  elementos estão sendo usado?
1. DIV: está sendo usada para poder fazer o menu das páginas.
2. a href: utilizado para linkar as páginas ao menu.
3. Target: dentro do a href para quando abrir a abar de uma página, consegui volta
4. H2: tá sendo utilizado para os títulos.
5. H4: para os extos.
6. Br: pras linhas ficarem embaixo uma da outra.
7. Small; utilizado para deixar o rodapé pequeno.
8. Footer: rodapé aonde diz de quem é o site.
9. Label: Utilizado para colocar legenda na interface do usuario.
10. Input: controles interativos para formulários e coleta os dados do usuario.
11. Img: para adicionar imagem.

# CSS : Quais propriedades estão sendo usado?
1. Font-family: para definir o estilo da fonte
2. Text alight: para colocar o texto no centro
3. Width: deinir o tamanho da imagem.
4. Color: adiciona cor ao texto.
5. background-color: definir cor do fundo
6. border-radius: criar uma borda
